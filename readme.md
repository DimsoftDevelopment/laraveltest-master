# Laravel project


## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

PHP v >= 7.2  
MySQL v >= 5.7  
Composer  
Git  


### Installing
Simply clone repo and run:
```
composer install
```
Configure .env

add POTTER_API_KEY (www.potterapi.com)
 
Then run 
```
php artisan migrate
php artisan db:seed

```

Enjoy!
